Welcome to Happy Seed

What's installed

Here is a basic middleman app that's setup with Bootstrap-SASS, jQuery, animate.css and socicons .

The following middleman components are installed

  • middleman
  • middleman-deploy
  • middleman-meta-tags
  • middleman-livereload
  • middleman-bootstrap-navbar
Read Middleman Documentation

What's configured

  • Layout and files moved to HAML
  • Bootstrap-Sass is included via Sass @import;
  • CNAME file gets generated if it exists for github pages subdomaining
  • data/site.yml file for default site meta data
  • meta_tags is enabled with the auto_meta tagger
  • Bootstrap nav bar helpers are actived
  • Livereload is on for development
  • Asset cleanup is on for build
  • Deploy to github pages setup by default
